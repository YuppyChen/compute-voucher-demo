# 算力券演示系统

一个用于展示算力券活动流程的HTML演示系统。

## 功能特性

- 📊 算力券活动状态机图（支持拖拽和缩放）
- ➕ 活动创建表单
- 📋 活动列表展示
- 📄 活动详情页面
- 🛒 算力券申领页面
- 📝 算力券申领表单
- ✅ 管理核销页面
- 🔍 审核页面

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 点击左侧导航栏切换不同页面
3. 所有页面都是静态展示，适合演示汇报

## 技术栈

- 纯 HTML + CSS + JavaScript
- 响应式设计
- 现代浏览器兼容
